# Phase 1: Golden Dataset 기반 파인튜닝 자동화
> 브랜치: `feature/issue-1045-adaptive-finetune`  
> 이슈 번호: [#1045] | 마일스톤: `v9.0 - Adaptive Intelligence` | 상태: 🏗️ 진행 중

## 🎯 목표
v8.0에서 자동 수집된 `golden_dataset`(.jsonl)을 활용하여 OpenAI Fine-tuning API를 통한 모델 고도화 파이프라인을 구축하고, 학습된 모델을 실제 챗 프레임워크에 동적으로 적용합니다.

---

## 📋 세부 작업 (Tasks)

### 1-1. 파인튜닝 서비스(FineTuningService) 엔진 개발
- [ ] `backend/services/finetune_service.py` 신설
- [ ] OpenAI API를 활용한 파일 업로드 및 Fine-tuning Job 생성 로직 구현
- [ ] 파인튜닝 진행 상태(Status) 폴링 및 데이터베이스 동기화

### 1-2. 모델 핫스왑(Hot-swap) 메커니즘 구현
- [ ] 학습 완료된 `ft-model-id`를 시스템 설정에 반영하는 관리자 API 구현 
  - (설계 원칙: **DB `AppConfig` 테이블** 기반 관리로 소스 코드 결합도 제거)
- [ ] LangGraph 에이전트 생성 시 최신 파인튜닝 모델을 우선 참조하도록 `nodes.py` 엔진 수정
- [ ] 모델 전환 시 기존 세션의 연속성 보장을 위한 캐시 무효화 전략 수립

### 1-3. 옵저버빌리티 및 성능 분석
- [ ] 파인튜닝 성공/실패 시 Discord 알림 봇 연동
- [ ] 이전 모델과 파인튜닝 모델 간의 성능 비교 분석 자동화 (User Rating 기반 점수 산출)
